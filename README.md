# Odoo Hackathon

Team repository for the Odoo Hackathon 2026 project.

## About

This repo holds our hackathon solution — code, docs, and project assets built during the event.

## Getting started

1. Clone the repository.
2. Review local team notes in `private/hackathonguidelines.md` (not tracked in git).
3. Install dependencies and run the app (instructions will be added as the stack is chosen).

## Team

- Up to 4 members per team
- Use Git collaboratively — everyone should commit, not just one person
